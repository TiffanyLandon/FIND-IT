# FIND-IT

## DESCRIPTION
FIND-IT allows users to buy and sell products via the FIND-IT market-place.

### THE FAST & EASY STEPS ARE:
* Sign-up
* Log-in
* Create a product in your dashboard, done.
* Buy a product easily displayed when logged-in.
